kbar - Assembly of Tangent Stiffness Matrix of bar element

FemLab Version 1.0, 04.05.95

Calling Sequence

K = kbar(K,T,X,G,u)
K = kbar(K,T,X,G)

Parameters

    Input:

  • K: existing global stiffness matrix
  • T: topology matrix for bar elements.
  • X: initial node coordinate matrix.
  • G: bar element property matrix.
  • u: displacement vector.

    • Output:

  • K: new global tangent stiffness matrix.

Description

Creates and assembles tangent stiffness matrix of elastic bar elements.
