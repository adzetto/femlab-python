Scilab Function

QET3P - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

[q,Se,Ee] = qet3p(Xe,Ge,Ue)

Parameters

    Input:

  • Xe: coordinates Xe = [x1 y1; x2 y2; x3 y3]
  • Ge: material parameters Ge = [k b] or Ge = k.
  • Ue: nodal potentials

    Output:

  • qe:element internal force vector.
  • Se: element flux (stress) vector.
  • Ee: element gradient (strain) vector.

Description

Evaluates gradient and flux for the current potentials. Creates the element internal force vector of a potential triangular 3-node element.
