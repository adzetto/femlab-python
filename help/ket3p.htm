Scilab Function

KeT3P - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

Ke = ket3p(Xe,Ge)

Parameters

    Input:

  • Xe: coordinates Xe = [x1 y1; x2 y2; x3 y3]
  • Ge: material parameters Ge = [k b] or Ge = k

    • Output:

  • Ke: element conductivity/dissipation matrix.

Description

Creates the combined element conductivity and dissipation matrix of potential triangular 3-node element
