Scilab Function

SetBC - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

[K,p,ks] = setbc(K,p,C) [K,p,ks] = setbc(K,p,C,dof) [K,p] = setbc(K,p,C) [K,p] = setbc(K,p,C,dof)

Parameters

    Input:

  • K: original global stiffness matrix.
  • p: global load vector.
  • C: constraint matrix, C = [ node1 dof1 u1
                                             node2 dof2 ...]
          or for dof=1,C=[ node1 u1
                                    node2 ...].
  • dof: number of dof pr. node.

    • Output:

  • K: global stiffness matrix with springs.
  • p: load vector including spring loads.
  • p: stiffness of constraining springs.

Description

Sets boundary conditions by imposing diagonal springs with 'large' stiffness ks.
