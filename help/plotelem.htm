Scilab Function

PlotElem - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence


plotelem(T,X)

plotelem(T,X,'linetype',nonum,noelem)

Parameters

    Input:

  • T: element topology matrix.
  • X: node coordinate matrix.
  • 'linetype':string defining linetype ('y-','c:',...).
  • nonum: if nonum = 1 -> node numbers will be plotted .
  • noelem: if noelem = 1 -> element numbers will be plotted .

    • Output:

  • K: new global system matrix.

Description

Plots elements in topology matrix T with coordinate matrix X. Uses linear line segment between all nodes.
