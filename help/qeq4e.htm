Scilab Function

QeQ4E - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

[qe, = qeq4e(Xe,Ge,Ue)

Parameters

    Input:

  • Xe:node coordinates.

    •       Xe = [x1 y1; x2 y2; x3 y3; x4 y4]
    .
  • Ge:element material data: [E , nu , (type)]

    •       optional: type = 1 : plane stress (default)
          type = 2 : plane strain
  • Ue: element displacement vector:

    •       Ue = [u1;v1;u2;v2;u3;v3;u4;v4]

    Output:

  • qe: internal force vector.
  • Se: element stress vector.
  • Ee: element strain vector.

Description

Evaluates the stress and strain for the current displacements. Creates the element internal force vector for elastic quadrilateral 4-node element in plane stress or plane strain. If Xe contains 8 node coordinate set the inter- nal force vector of 8-node element with quadratic shape functions is evaluated using reduced integration.
