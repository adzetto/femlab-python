Scilab Function

KT3E - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

K = kt3e(K,T,X,G)

Parameters

    Input:

  • K: initial global stiffness matrix.
  • T: element topology matrix.
  • X: node coordinate matrix.
  • G: material property matrix.

    • Output:

  • K: new global stiffness matrix.

Description

Creates and assembles stiffness matrix for a group of elastic triangular 3-node elements in plane stress or plane strain.
