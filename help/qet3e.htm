Scilab Function

QeT3E - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

[qe,Se,Ee] = qet3e(Xe,Ge,Ue)

Parameters

    Input:

  • Xe: corner coordinates (3 rows).
  • Ge: element material data: [E , nu , (type)].

    •       optional : type = 1 : plane stress (default)
                         type = 2 : plane strain
  • Ue: element displacement vector: [u1;v1;u2;v2;u3;v3]

    Output:

  • qe: internal force vector.
  • Se: element stress vector.
  • Ee: element strain vector.

Description

Evaluates the stress and strain for the current displacements. Creates the element internal force vector for elastic triangular 3-node element in plane stress or plane strain.
