assmq – assmq matrix

Calling Sequence 

   q = assmq(q,qe,Te)

   q = assmq(q,qe,Te,dof)

 

 

 

Parameters

 

Input:

   q  :  global vector.

   qe :  element vector.

   Te :  element topology vector.

   dof:  degrees of freedom per node.

 

Output:

   q  :  updated global vector.

 

 

Description

      Assembles global force vector by adding element

   nodal forces to existing global force vector.
