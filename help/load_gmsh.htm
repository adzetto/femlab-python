Scilab Function

load_gmsh - Load mesh data from Gmsh output file

FemLab Version 1.0, 04.05.95

Calling Sequence


gmesh = load_gmsh(filename)

Parameters

    Input:

  • filename: Filename of the mesh data (output of Gmsh Version 1.0)

    • Output:

  • gmesh: structure variable
  • gmesh.nbNod: gives the number of nodes
  • gmesh.LINES: gives the topology matrix of line elements.
  • gmesh.QUAD: gives the topology matrix of quad elements.

  • Type gmesh to see all available data.

Description

Reads the mesh output file of the pre- and post processor, GMSH, which is freely available at http://www.geuz.org/gmsh .
