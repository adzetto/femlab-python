Scilab Function

QT3E - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

[q,S,E] = qt3e(q,T,X,G,u)

Parameters

    Input:

  • q: initial global force vector.
  • T: element topology matrix.
  • X: node coordinate matrix.
  • G: material property matrix.
  • u: global displacement vector.

    • Output:

  • q: new global internal force vector.
  • S: global stress matrix.
  • E: global strain matrix.

Description

Creates and assembles internal force vector, stress matrix and strain matrix for a group of elastic triangular 3-node elements in plane stress or plane strain.
