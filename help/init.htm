Scilab Function

init - Initializes global matrices and vectors

FemLab Version 1.0, 04.05.95

Calling Sequence

K = init(nn,dof)
[K,p] = init(nn,dof)
[K,p,q] = init(nn,dof)

Parameters

    Input:

  • nn:number of global nodes.
  • dof: number of degrees of freedom per node.

    • Output:

  • K: initialized global stiffness matrix.
  • p: initialized global load vector.
  • q: initialized global internal force vector.

Description

Initializes global stiffness matrix, global load vector and global internal force vector.
