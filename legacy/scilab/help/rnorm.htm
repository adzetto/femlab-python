Scilab Function

rnorm - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

n = rnorm(f,C,dof)

Parameters

    Input:

  • f: force vector.
  • C: constraint matrix.
  • dof:number of dof pr. element node.

    • Output:

  • n: Reduced Euclidian norm of f.

Description

Evaluates the reduced Euclidian norm of the vector, f. Constrained terms in f are identified from the matrix C specifies the prescribed dof.
