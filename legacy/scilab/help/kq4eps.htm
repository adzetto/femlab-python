KQ4EPS - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

K = kq4eps(K,T,X,G,S,E) K = kq4eps(K,T,X,G,S,E,type)

Parameters

    Input:

  • K: initial global stiffness matrix.
  • T: element topology matrix.
  • X: node coordinate matrix.
  • G: material property matrix.
  • S: current stress matrix.
  • E: current strain matrix.
  • mtype: parameter setting material model
          mtype = 1 -> Von Mises
          mtype = 2 -> Drucker-Prager

    Output:

  • K: new global tangent stiffness matrix.

Description

Creates and assembles tangent stiffness matrix for a group of elasto-plastic quadrilateral 4-node elements in plane stress.
