Scilab Function

KeT3E - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

Ke = ket3e(Xe,Ge)

Parameters

    Input:

  • Xe: corner coordinates (3 rows).
  • Ge: element material data: [E , nu , (stype)].

    •       optional : stype = 1 : plane stress (default
                         stype = 2 : plane strain

    Output:

  • Ke: element stiffness matrix.

Description

InputCreates the element stiffness matrix of elastic triangular 3-node element in plane stress or plane strain.
