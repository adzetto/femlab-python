solve_lag - Solution of the linear system with Boundary Conditions

FemLab Version 1.0, 04.05.95

Calling Sequence

u = solve(K,p)
u = solve(K,p,C,dof)

Parameters

    Input:

  • K : global matrix
  • p : global load vector.
  • C : Constraint Matrix.
  • dof : degrees of freedom per node.

    • Output:

  • K : updated global matrix.

Description

Solves the sytem matrices to obtain the Global structural displacement vector by using the Lagrange Multiplier Method.
