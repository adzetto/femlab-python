Scilab Function

QeBar - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

qe = qebar(Xe0,Xe1,Ge)

Parameters

    Input:

  • Xe0:initial coordinates = [x1 y1 (z1)
    x2 y2 (z3) ].
    • .
  • Xe1: current coordinates = [x1+u1 y1+v1 (z1+w1)
    x2+u2 y2+v2 (z3+w2)].
    • .
  • Ge: element properties Ge = [A E] or Ge = AE.

    • Output:

  • qe: element nodal force vector.
  • Se: element axial stress.
  • Ee: element axial strain.

Description

Evaluates the strain and stress for the current displacements. Creates the element internal force vector for elastic bar element.
