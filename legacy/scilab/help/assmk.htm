Scilab Function

assmk - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

K = assmk(K,Ke,Te)
K = assmk(K,Ke,Te,dof)

Parameters

    Input:

  • K: global matrix
  • Ke: element matrix.
  • Te: element topology vector.
  • dof: degrees of freedom per node.

    • Output:

  • K: updated global matrix.

Description

Assembles system matrix by adding element matrix to existing global matrix. The system matrix may be a stiffness matrix, mass matrix, conductivity matrix, etc.
