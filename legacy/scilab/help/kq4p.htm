Scilab Function

KQ4P - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

K = kq4p(K,T,X,G)

Parameters

    Input:

  • K:initial global system matrix.
  • T: element topology matrix.
  • X: node coordinate matrix.
  • G: material property matrix.

    • Output:

  • K: new global system matrix.

Description

Creates and assembles combined conductivity and dissipation matrix for a group of potential quadrilateral 4-node elements.
