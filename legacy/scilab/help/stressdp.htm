Scilab Function

stressdp - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

[S,dL] = stressdp(S,G,Sy,dE,dS)

Parameters

    Input:

  • S: elastic predictor stress = [ S11
                                                S22
                                                S12 ]
  • G: material properties = [ E nu S0 H ]
  • Sy: current (intermediate) yield stress
  • dE:
  • dS:

    Output:

  • S: updated stress vector.
  • dL: plastic multiplier increment.

Description

Calculates the updated stress and plastic multiplier of used in a backward Euler algorithm for associated Drucker-Prager materials in plane stress.
