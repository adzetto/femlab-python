Scilab Function

QT3P - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

[q,S,E] = qt3p(T,X,G,u)

Parameters

    Input:

  • T: element topology matrix.
  • X: node coordinate matrix.
  • G: material property matrix.
  • u: global potential vector.

    • Output:

  • q: internal force vector.
  • S: global flux matrix.
  • E: global gradient matrix.

Description

Creates and assembles internal force vector, gradient matrix and flux matrix for a group of potential triangular 3-node elements.
