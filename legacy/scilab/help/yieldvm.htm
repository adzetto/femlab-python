Scilab Function

yield - Assembly of System Matrix

FemLab Version 1.0, 04.05.95

Calling Sequence

f = yieldvm(S,G,dL,Sy)

Parameters

    Input:

  • S: elastic predictor stress = [ S11
                                                S22
                                                S12 ]
  • G: material properties = [ E nu S0 H ]
  • dL: plastic multiplier increment.
  • Sy: current (intermediate) yield stress.

    • Output:

  • f: value of yield function.

Description

Calculates the updated stress and plastic multiplier of used in a radial return algorithm for von Mises materials in plane stress.
